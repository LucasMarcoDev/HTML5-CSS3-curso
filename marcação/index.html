Abreviações e inversão de palavras

Em HTML5 podemos também usar as tags para fazermos abreviações usando a tag abbr exemplo: HTML5 e CSS3 onde html e css estão envolvidos nessas tags que mostratrao o nome de suas abreviações no site. Seguindo temos também a tag bdo que faz com que uma palavra fique escrita ao contrário como a essa próxima palavra algodão que está envolvida pela tag bdo a palavra é algodao mas como esta envolvida pela tag irá aparecer ao contrário para quem olha o site.

citações

Em HTML5 podemos fazer citações usando a tag q, por exemplo: um dia todos nós seremos mais mais fortes onde utilizando essa tag o HTML5 indicará a citação colocando aspas na parte que a tag for inserida. Podemos também fazer citações por blocos utilizando a tga blockquote onde o HTML5 irá pular uma linha criar um espaçamento e colocar a citação em uma espécie de bloco como nesse exemplo: Seu Madruga disse:

as pessoas boas devem amar seus inimigos
e dentro da tag blockquote podemos colocar o parametero cite para indicarmos ao HTML5 de onde vem a citação utilizada

formatação
